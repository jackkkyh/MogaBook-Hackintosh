# MogaBook-Hackintosh
Hackintosh EFI for Lenovo Yoga 710-14ikb. 

### Bootloader: `Clover` and `OpenCore`

### OS: macOS Catalina
  > **macOS Big Sur** throws `disk write locked` error
### Specs:
* i5-7200U (HD 620)
* DW1820A

### Features:
- [x] Graphics & fine-tuned brightness adjustment
  > use Fn+C and Fn+P
- [x] Wi-Fi & bluetooth
- [x] AirDrop & Handoff
- [x] Sleep
- [ ] Touchscreen
