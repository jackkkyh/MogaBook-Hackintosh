# MogaBook-Hackintosh
Hackintosh EFI for Lenovo Yoga 710-14ikb. 

### Specs:
* i5-7200U (HD 620)
* DW1820A

### Bootloader: `Clover` and `OpenCore`

### OS: `macOS Catalina`
- model: MacBookPro14,1 
  
### Features:
- [x] Graphics & fine-tuned brightness adjustment
  > use Fn+C and Fn+P
- [x] Wi-Fi & bluetooth
- [x] AirDrop & Handoff
- [x] Sleep
- [ ] Touchscreen
- [ ] `disk write locked` error on **macOS Big Sur**
